## 功能模块关系

```
组织管理
  ├── 专业管理（专业属于组织）
  │     └── 班级管理（班级属于专业）
  │           └── 学生管理（学生属于班级）
  │
  └── 教师管理（教师属于组织/专业）
        └── 课程管理（教师授课）

数据字典管理（为所有模块提供基础数据）
财务管理（管理学生缴费，关联学生和专业）
课程封面管理（为课程提供封面图片）
LLM管理（为AI功能提供配置）
```

## 通用功能

所有管理模块都包含以下通用功能：

1. **搜索功能**：支持按不同条件搜索和筛选数据
2. **批量导入**：支持通过Excel文件批量导入数据
3. **数据展示**：以表格形式展示数据列表
4. **操作功能**：编辑、删除、查看详情等操作
5. **状态管理**：激活/停用状态切换
