# 专业管理功能说明文档

## 功能概述

**功能名称**：专业学科建设  
**访问路径**：`/admin/majors`  
**功能描述**：管理学院专业设置、培养方案与学费

## 页面布局

### 页面标题
- **主标题**：专业学科建设
- **副标题**：管理学院专业设置、培养方案与学费

### 功能按钮
1. **批量导入**：支持批量导入专业数据
2. **添加新专业**：创建新的专业

### 搜索功能
- **专业名称**：输入框，支持按专业名称搜索
- **搜索按钮**：执行搜索操作
- **重置按钮**：清空搜索条件，显示全部数据

### 统计信息卡片
页面顶部显示4个统计指标：
- **专业总数**：当前系统中的专业总数量
- **平均学费/年**：所有专业的平均年度学费
- **平均学制(年)**：所有专业的平均学制年限（默认4年）
- **所属院系**：专业所属的组织/院系数量

### 数据表格
表格显示以下列：
- **专业名称**：专业的名称
- **描述**：专业的详细描述信息
- **所属组织**：专业所属的组织/院系
- **学费标准**：专业的年度学费金额
- **学制**：专业的学制年限（如4年）
- **操作**：对专业进行编辑、删除等操作

### 空状态提示
当没有数据时，显示：
- 图标提示
- 文字："暂无数据"
- 提示："点击右上角'添加新专业'按钮创建专业"

## 核心功能

### 1. 添加新专业
**触发方式**：点击右上角"添加新专业"按钮

**表单字段**（推测）：
- **专业名称**（必填）：输入专业的名称
- **所属组织**（必填）：选择专业所属的组织/院系
- **学费标准**（必填）：输入年度学费金额
- **学制**（必填）：选择或输入学制年限（年）
- **描述**（可选）：输入专业的详细描述

**操作按钮**：
- **取消**：关闭弹窗，不保存
- **保存**：保存专业信息

### 2. 批量导入
**触发方式**：点击右上角"批量导入"按钮

**功能说明**：支持通过Excel文件批量导入专业数据

### 3. 搜索功能
- **按专业名称搜索**：输入专业名称关键词进行模糊搜索
- **重置搜索**：清空搜索条件，恢复显示全部数据

### 4. 数据展示
- 以表格形式展示所有专业信息
- 显示专业的学费、学制等关键信息
- 支持查看专业所属的组织信息

### 5. 操作功能
表格中每行记录的操作列可能包含：
- 编辑：修改专业信息
- 删除：删除专业（需确认）
- 查看详情：查看专业详细信息

## 使用场景

1. **专业设置**：学校新增专业时，创建新的专业记录
2. **学费管理**：设置和调整各专业的学费标准
3. **培养方案**：管理专业的学制和培养方案
4. **专业查询**：通过搜索功能快速查找专业信息
5. **数据统计**：查看专业的整体统计信息

## 注意事项

1. 专业必须关联到某个组织/院系
2. 学费标准为年度费用，单位为人民币（¥）
3. 学制通常为4年，但可根据实际情况调整
4. 删除专业前需要确认，避免误操作
5. 专业下有关联数据（班级、学生）时，删除操作可能需要特殊处理
