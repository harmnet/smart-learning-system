# 教师管理功能说明文档

## 功能概述

**功能名称**：教师团队管理  
**访问路径**：`/admin/teachers`  
**功能描述**：管理师资力量、教学任务分配

## 页面布局

### 页面标题
- **主标题**：教师团队管理
- **副标题**：管理师资力量、教学任务分配

### 功能按钮
1. **批量导入**：支持批量导入教师数据
2. **添加新教师**：创建新的教师账号

### 搜索功能
- **教师姓名**：输入框，支持按教师姓名搜索
- **手机号**：输入框，支持按手机号搜索
- **状态**：下拉选择框，按状态筛选
  - 选项："全部"（默认）、"激活"、"停用"
- **搜索按钮**：执行搜索操作
- **重置按钮**：清空搜索条件，显示全部数据

### 数据表格
表格显示以下列：
- **教师姓名**：教师的真实姓名
- **邮箱**：教师的邮箱地址
- **手机号**：教师的手机号码（通常作为登录账号）
- **所属组织**：教师所属的组织/院系
- **所属专业**：教师所属的专业
- **状态**：教师账号状态（激活/停用）
- **操作**：对教师进行编辑、删除等操作

## 核心功能

### 1. 添加新教师
**触发方式**：点击右上角"添加新教师"按钮

**表单字段**（推测）：
- **姓名**（必填）：输入教师的真实姓名
- **手机号**（必填）：输入教师的手机号码（作为登录账号）
- **邮箱**（可选）：输入教师的邮箱地址
- **所属组织**（必填）：选择教师所属的组织/院系
- **所属专业**（可选）：选择教师所属的专业
- **密码**（必填）：设置教师的初始登录密码
- **状态**（默认）：激活

**操作按钮**：
- **取消**：关闭弹窗，不保存
- **保存**：保存教师信息

### 2. 批量导入
**触发方式**：点击右上角"批量导入"按钮

**功能说明**：支持通过Excel文件批量导入教师数据

### 3. 搜索功能
- **按教师姓名搜索**：输入教师姓名关键词进行模糊搜索
- **按手机号搜索**：输入手机号进行搜索
- **按状态筛选**：通过下拉选择框筛选激活或停用的教师
- **组合搜索**：可以同时使用多个搜索条件
- **重置搜索**：清空所有搜索条件，恢复显示全部数据

### 4. 数据展示
- 以表格形式展示所有教师信息
- 显示教师的基本信息和账号状态
- 支持查看教师所属的组织和专业信息

### 5. 操作功能
表格中每行记录的操作列可能包含：
- 编辑：修改教师信息
- 删除：删除教师账号（需确认）
- 查看详情：查看教师详细信息
- 重置密码：重置教师的登录密码
- 修改状态：激活或停用教师账号
- 分配课程：为教师分配教学课程

## 使用场景

1. **教师注册**：为新入职教师创建账号
2. **教师查询**：通过多种条件快速查找教师
3. **信息维护**：更新教师的个人信息
4. **账号管理**：管理教师的账号状态和权限
5. **批量操作**：通过批量导入快速添加大量教师

## 注意事项

1. 手机号通常作为教师的登录账号，需要唯一
2. 教师可以关联到某个组织和专业
3. 删除教师前需要确认，避免误操作
4. 教师有教学记录时，删除操作可能需要特殊处理
5. 批量导入需要按照指定格式准备Excel文件
6. 停用教师账号不会删除数据，只是禁止登录
