# 班级管理功能说明文档

## 功能概述

**功能名称**：班级教务管理  
**访问路径**：`/admin/classes`  
**功能描述**：管理班级建制、课程安排与学生归属

## 页面布局

### 页面标题
- **主标题**：班级教务管理
- **副标题**：管理班级建制、课程安排与学生归属

### 功能按钮
1. **批量导入**：支持批量导入班级数据
2. **创建新班级**：创建新的班级

### 搜索功能
- **班级名称**：输入框，支持按班级名称搜索
- **所属组织**：下拉选择框，按组织筛选班级
  - 选项："请选择"（默认）
- **所属专业**：下拉选择框，按专业筛选班级
  - 选项："请选择"（默认）
- **搜索按钮**：执行搜索操作
- **重置按钮**：清空搜索条件，显示全部数据

### 数据表格
表格显示以下列：
- **班级名称**：班级的名称
- **班级代码**：班级的唯一标识代码
- **所属组织**：班级所属的组织/院系
- **所属专业**：班级所属的专业
- **年级**：班级的年级信息
- **人数**：班级中的学生人数
- **操作**：对班级进行编辑、删除等操作

## 核心功能

### 1. 创建新班级
**触发方式**：点击右上角"创建新班级"按钮

**表单字段**（推测）：
- **班级名称**（必填）：输入班级的名称
- **班级代码**（必填）：输入班级的唯一标识代码
- **所属组织**（必填）：选择班级所属的组织/院系
- **所属专业**（必填）：选择班级所属的专业
- **年级**（必填）：输入或选择年级（如2024）
- **学期**（可选）：选择学期信息

**操作按钮**：
- **取消**：关闭弹窗，不保存
- **保存**：保存班级信息

### 2. 批量导入
**触发方式**：点击右上角"批量导入"按钮

**功能说明**：支持通过Excel文件批量导入班级数据

### 3. 搜索功能
- **按班级名称搜索**：输入班级名称关键词进行模糊搜索
- **按所属组织筛选**：通过下拉选择框筛选特定组织下的班级
- **按所属专业筛选**：通过下拉选择框筛选特定专业下的班级
- **组合搜索**：可以同时使用多个搜索条件
- **重置搜索**：清空所有搜索条件，恢复显示全部数据

### 4. 数据展示
- 以表格形式展示所有班级信息
- 显示班级的基本信息和学生人数
- 支持查看班级所属的组织和专业信息

### 5. 操作功能
表格中每行记录的操作列可能包含：
- 编辑：修改班级信息
- 删除：删除班级（需确认）
- 查看详情：查看班级详细信息
- 管理学生：查看和管理班级学生

## 使用场景

1. **班级创建**：每学年创建新班级
2. **班级查询**：通过多种条件快速查找班级
3. **学生归属**：管理学生与班级的关联关系
4. **课程安排**：为班级安排课程和教师
5. **数据统计**：查看各班级的学生人数等信息

## 注意事项

1. 班级必须关联到某个组织和专业
2. 班级代码应该是唯一的标识符
3. 删除班级前需要确认，避免误操作
4. 班级下有学生时，删除操作可能需要特殊处理（如转移学生）
5. 年级信息通常与入学年份相关
